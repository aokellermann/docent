### Development environment

- We recommend using the manual installation in [quickstart](../quickstart.md) for more direct access to the code. You can still use Docker for the databases.
- If you use VSCode/Cursor/Windsurf, we recommend using the `.vscode/docent.code-workspace` workspace, which comes with useful default settings.
